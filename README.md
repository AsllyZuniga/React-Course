## 📖 Descripción

Este repositorio contiene los proyectos y ejercicios del **curso completo de React**, diseñado para llevar al estudiante desde los conceptos más básicos hasta la construcción de aplicaciones full-stack con el **stack MERN**.

A lo largo del curso aprenderás a:
- 🏗️ Construir aplicaciones reales con React y TypeScript
- 🧪 Escribir pruebas unitarias e integración
- 🔐 Implementar autenticación con Google y JWT
- 🗄️ Manejar estado global con **Redux Toolkit** y **Context API**
- 🌐 Desarrollar APIs REST con Node.js, Express y NestJS

## 🚀 Cómo Empezar

**1. Clona el repositorio**
```bash
git clone https://github.com/AsllyZuniga/React-Course.git
cd React-Course
```

**2. Entra al módulo que quieras explorar**
```bash
cd 01-intro-react
```

**3. Instala dependencias y levanta el proyecto**
```bash
npm install
npm run dev
```

> ⚠️ Algunos módulos (10 y 12) requieren configurar variables de entorno. Revisa el archivo `.env.example` dentro de cada carpeta.
